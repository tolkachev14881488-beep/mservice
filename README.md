## Локально

```bash
npx serve .
```

## Домен mservice.by

1. GitHub → репозиторий → **Settings** → **Pages** → Custom domain: `mservice.by`
2. У регистратора DNS: `CNAME mservice.by` → `tolkachev14881488-beep.github.io`
